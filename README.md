# Fruit and Vegetable Ordering System

This is a beginner-friendly web-based project that allows users to view fruits and vegetables and place simple orders online.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Display fruits and vegetables
- Simple order form
- Responsive layout
- Easy to understand for beginners

## Purpose
This project is created for learning web development basics including HTML structure, CSS styling, and JavaScript interaction.
